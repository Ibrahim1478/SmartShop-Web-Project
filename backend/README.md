# SmartShop Backend (Spring Boot)

## Run in IntelliJ
Open this folder as a Maven project and run:
`SmartShopApplication`

## API
- GET `http://localhost:8080/api/products`
- GET `http://localhost:8080/api/products/featured`
- GET `http://localhost:8080/api/products/{id}`

## H2 console
`http://localhost:8080/h2-console`
JDBC URL: `jdbc:h2:mem:smartshopdb`
User: `sa` (no password)
